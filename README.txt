Portfolio Fix Pack (Light Theme + Banners + Resume)
====================================================

This pack fixes your three requests:
1) Adds a visible portfolio banner on the homepage (assets/banners/portfolio-banner.svg)
2) Adds a Resume button (points to /Srikanth_Madabhushi_Resume.pdf)
3) Uses LinkedIn article banners for EVERY project (replace the SVG placeholders in /assets/banners/ with your JPG/PNG banner images)

Files:
- index.html         (homepage, light theme by default, banner + Resume link)
- projects.html      (projects, uses /assets/banners/* for each card, light theme by default)
- assets/banners/portfolio-banner.svg
- assets/banners/{itsm1.svg, itsm2.svg, itsm3.svg, itsm4.svg, itom1.svg, itom2.svg, itom4.svg, grc1.svg, grc2.svg, grc3.svg, csm1.svg, csm2.svg, csm3.svg, hrsd1.svg, hrsd2.svg, hrsd3.svg}

How to use:
1) Upload everything to your repo root (srikanthmadabhushi.github.io), keeping folders/paths.
2) If your resume file name is different, rename your PDF to Srikanth_Madabhushi_Resume.pdf (or update the link in index.html).
3) Replace banner placeholders with your real LinkedIn banner images:
   - e.g., upload /assets/banners/itsm1.jpg and then update projects.html to use .jpg instead of .svg, or keep the same filenames but overwrite the .svg files with your actual banners (SVG/PNG/JPG all work).
4) Visit:
   - Home:     https://srikanthmadabhushi.github.io/
   - Projects: https://srikanthmadabhushi.github.io/projects.html
5) Use the Theme toggle if you prefer dark mode.

Notes:
- The background uses a bright, clean palette by default (no pure black).
- All images are responsive 16:9 banners (recommended size ~1600x900).
